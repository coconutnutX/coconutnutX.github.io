Unity Debug WebGL播放视频

2021-04-02
Note

Unity内运行时播放视频没问题，导出后用火狐打开就放不了，看控制台报错是

解决方法

  1. WebGL只能通过URL播放视频

    Unity2019在WebGL下播放视频

  2. 火狐浏览器设置允许跨域

    security.fileuri.strict_origin_policy改false

    Firefox火狐浏览器访问本地文件-提示跨域问题

  3. 火狐浏览器设置允许播放视频

    Unity2018发布webgl视频无法播放
